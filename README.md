# Car Price Prediction with Machine Learning

Predicts car price using features such as year, mileage, fuel type, transmission, etc.

## Files
- `Car_Price_Prediction.ipynb` — Colab notebook with data preprocessing, training, and evaluation
- `car data.csv` — dataset used

## How to run
1. Open `Car_Price_Prediction.ipynb` in Google Colab or Jupyter.
2. Install required packages (if needed): `pip install pandas scikit-learn matplotlib seaborn`
3. Run cells top-to-bottom.
